Three ways to use each signature, in order of convenience:

  1. Copy signature → paste into Gmail (Settings → Signature) or desktop Outlook. Works everywhere except mobile / new Outlook, which strip embedded images.
  2. Copy HTML source → in your signature editor tap the HTML / source-code button, then paste. The source view skips the HTML sanitizer that drops the logo.
  3. Download .png → a flat image of the whole signature. Use your editor's Insert image button if the copy-paste routes both fail.
Theme-agnostic. No background is copied. The mark + text use gold (#C9A96E) and medium grey (#6A6A6A) — both read on light and dark email clients.
